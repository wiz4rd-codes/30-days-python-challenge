## 💻 Projects

### 1. Login Protected Product Dashboard
A simple dashboard protected using a custom decorator.

**Features**
- Checks login before opening the dashboard
- Displays all products
- Filters products costing ₹1000 or more
- Calculates total inventory value using `reduce()`

---

### 2. Movie Rating Management System
A movie management program using Python properties.

**Features**
- Add multiple movies
- Store movie names and ratings
- Update ratings using `@property` and `@setter`
- Validate ratings (0–10)
