# Day 13 - 30 Days Python Challenge 🐍

## What I Learned Today
- Dunder (Magic) Methods
- Method Overriding

## Programs Written

### 1. Book System
A program that manages books and demonstrates dunder methods — `__str__` for
custom print output, `__len__` for page count, `__gt__` for price comparison,
and `__eq__` for checking if two books are the same.

### 2. Product & Discount Product System
A program that manages products, where `DiscountProduct` inherits from `Product`
and overrides the `total_price()` method to apply a discount, while still
reusing the parent's logic using `super()`.

## Concepts Covered
- `__str__`, `__len__`, `__gt__`, `__eq__` dunder methods
- Operator overloading using dunder methods
- Method overriding in child classes
- Using `super()` inside an overridden method
- Object list management with `enumerate()`
