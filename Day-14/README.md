# Day 14 - 30 Days Python Challenge 🐍

## What I Learned Today
- Operator Overloading
- Working with external libraries (pypdf)

## Programs Written

### 1. PDF Merger
A program that merges multiple PDF files into a single PDF using the `pypdf`
library's `PdfWriter` class.

### 2. Cricketer System
A program that manages cricketer stats and demonstrates operator overloading —
`__gt__` to compare runs, `__eq__` to check equality, `__len__` to return
matches played, and `__add__` to combine two players into a new merged player.

## Concepts Covered
- Operator overloading (`__add__`, `__gt__`, `__eq__`, `__len__`)
- Using external libraries (`pypdf`)
- Object comparison and combination
- Object list management with `enumerate()`
